## Деплой файл

**Файл:** `deploy.ps1`

**Описание:** PowerShell скрипт для автоматического деплоя на сервер

**Использование:**
```powershell
cd C:\tg
.\deploy.ps1
```

**Что делает:**
1. Добавляет все изменения в Git
2. Создает коммит с текущей датой/временем
3. Отправляет изменения на сервер через SSH
4. На сервере автоматически обновляется сайт без прерывания работы

## Структура проекта

Основные файлы:
- `index.html` - главная страница сайта
- `style.css` - стили
- `app.js` - Node.js API сервер
- `package.json` - зависимости Node.js
- `nginx_tg-text.ru.conf` - конфигурация Nginx
- `post-receive` - Git hook для автоматического деплоя на сервере
- `deploy.ps1` - скрипт деплоя (Windows PowerShell)
- `setup_api.sh` - скрипт настройки API на сервере
- `text-processor.service` - systemd сервис для Node.js API
